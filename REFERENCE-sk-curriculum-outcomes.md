# Saskatchewan K-12 Curriculum - Complete Learning Outcomes Reference

**966 Learning Outcomes across 13 Grade Levels**
**7 Subject Areas:** English Language Arts, Mathematics, Science, Social Studies, Health Education, Physical Education, Arts Education
**Source:** Saskatchewan Ministry of Education — curriculum.gov.sk.ca

---

## Purpose

This reference file maps all Saskatchewan curriculum outcomes so that Wolf Whale LMS mini games, interactive tools, and educational content can be directly linked to specific learning outcomes. Each game/tool will target one or more outcome codes.
